# Operator Guide

## First Deployment
1. Register operator, terminals/berths, safety and hazmat-handling
   scope, staff and robots.
2. Import existing consignment-booking and billing history.
3. Run read-only safety-scope and crane/reach-stacker/forklift robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run reconciliation record and audit export.

## Minimum Production Controls
- terminal safety-scope and hazmat-scope validation before any
  handling dispatch
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (out-of-scope
  handling, out-of-scope hazmat lift, uninspected lift plan)
- evidence-backed reconciliation records
- audit export for every dispatch, sign-off and reconciliation record
- backup manual cargo-handling process

## Certification
Certified operators must prove robot-safety integrity, terminal
safety and hazmat-scope discipline, evidence-backed reconciliation
records and human review for handling-affecting actions.
